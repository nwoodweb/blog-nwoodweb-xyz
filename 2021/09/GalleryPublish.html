Workflow for Publishing to Gallery

Compression

It is in our interest to save server storage space, using our VPS it is best to choose a lossy file format, with my preference being JPEG

If we are storing on a workstation or a NAS, it is in our interest to store our images in lossless raster formats or extensible vector formats


I use the Gnu Image Manipulation Program (GIMP) to first compress the lossless files at maximum compresion, then reimport the compressed lossless file to then export in JPEG with Compression Ratio = 50
