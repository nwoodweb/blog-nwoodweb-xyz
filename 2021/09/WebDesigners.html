Good Sources for Web Design and Server Setup

The Documentation of Course!

It is always a good idea to read up on the documentation of the software you are using, be it a website, wiki, or the man pages. Even the --help or -h flags are helpful.

W3 Schools Website

  • BootStrap
  • Python
  • R

Roman Zolotarev

Roman Zolotarev's Website

Roman Zolotarev is a professional web developer in Europe, and provides basic tutorials with regards to using OpenBSD as a server and desktop

I believe he teaches a paid course for people, but I have never participated

UX Planet

A series of blogs that discuss user experience design in a more broad sense

World Wide Web Consortium

The W3C is the main international standards organization for the internet, and collectively defines standards for protocols such as MathML, HTML/CSS, and privacy.
